# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

This is a **static site** for the Santa Monica Tennis Club (www.santamonicatennisclub.net), hosted on GitHub Pages. The HTML files in this repo are edited directly and are the source of truth.

Deployed domain is set in `CNAME`. GitHub Pages serves everything from the `main` branch.

## Deployment workflow

Edit `.html` files directly, then commit and push — GitHub Pages auto-deploys.

## Architecture

- The site was originally generated by **WordPress + Elementor**, which explains the structure of the HTML
- **Modern Events Calendar Lite** powers the events/calendar section
- URL structure maps directly to directory structure (`/contact/` → `/contact/index.html`)
- All CSS, JS, and media assets live under `/wp-content/`
- Inline styles (CSS variables for colors, fonts, spacing) appear in each page's `<head>`
